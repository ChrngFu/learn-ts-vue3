# Vue 3 + TypeScript + Vite

这个模板可以帮助你开始在 Vite 中使用 Vue 3 和 TypeScript 进行开发。该模板使用 Vue 3 的 `<script setup>` 单文件组件，请查看 [script setup 文档](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) 了解更多内容。

在 [Vue 文档 TypeScript 指南](https://vuejs.org/guide/typescript/overview.html#project-setup) 中了解更多关于推荐的项目设置和 IDE 支持的内容。

此项目基于当前时间(2025-11-03)最新的 Vue 3、TypeScript 和 Vite 7 构建。

## 安装依赖
```bash
  npm install
```

## 运行项目

```bash
  npm run dev
```

## 其他
> 1. 运行项目时，请确保已安装 Node.js 和 npm。
> 2. 此项目推荐node.js版本为 20+,22+
